# Arweave Web Wallet

## Install

```sh
npm i && cd src/popup && npm i
```

## Run with dev server

```sh
npm run watch
```

## Build

```sh
npm run build
```

## Install locally

1. Extract extension.zip to any directory.
2. Open Chrome/Brave and go to [](chrome://extensions)
3. Click "Load unpacked extension" and open the "_extension-directory_/dist" directory
